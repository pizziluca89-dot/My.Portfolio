# My.Portfolio
This repository contains the source code of my personal portfolio website, built to clearly and professionally showcase my work as a psychotherapist, trainer, and full-stack developer in training.

The website includes:

A homepage with a hero section and professional introduction

Experience, Education, and Services sections

A contact system powered by EmailJS

Responsive layout built with Bootstrap 5

Optional dark/light mode support

Fully optimized for deployment via GitHub Pages

The goal of this project is to provide a modern and accessible online platform to present my professional background and skills.

Questo repository contiene il codice sorgente del mio sito portfolio personale, sviluppato per presentare in modo chiaro e professionale la mia attività come psicoterapeuta, formatore e full-stack developer in formazione.
